===================================================================
🤖 COMMANDS FOR BOTFATHER
===================================================================
Copy and paste these commands into BotFather when configuring your bot:
(Main Menu → /mybots → Choose your bot → Edit Bot → Edit Commands)

status - 📊 Quick system overview
quick - ⚡ Ultra-compact one-liner status
docker - 🐳 Manage Docker containers
dstats - 📈 Container resource usage
top - 🔝 Top processes by CPU/RAM
temp - 🌡 Check system temperatures
net - 🌐 Network information
logs - 📝 View recent system logs
logsearch - 🔍 Search container logs
report - 📋 Generate full system report
ping - 🏓 Check bot availability
health - 💓 Healthchecks.io status & stats
config - ⚙️ Show current configuration
sysinfo - 💻 Detailed system information
speedtest - 🚀 Run internet speed test
diskpred - 💾 Disk space prediction
settings - 🔧 Configure bot settings
language - 🌍 Change language (EN/IT)
container - 📦 Manage specific container
kill - 💀 Force kill a container
restartdocker - 🔄 Restart Docker service
update - ⬇️ Download and install latest update
changelog - 📄 Show the latest release changelog
reboot - 🔁 Reboot the system
shutdown - ⏹ Shutdown the system
help - ❓ Show all available commands
